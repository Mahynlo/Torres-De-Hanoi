# Torres-De-Hanoi
Juego de las Torres de Hanoi 

## Reglas del juego
El juego consiste en una base con 3 estacas verticales. En la primera se ubica un numero determinado de discos ordenados de el de radio menor al de radio mayor, siendo el de mas arriba el de radio menor.
El objetivo del juego es pasar los discos de la primera estaca a la tercera estaca con las siguientes restricciones:
Solo mover un disco a la vez.
Un disco solo se puede colocar sobre otro disco mas grande.

## Programa
El objetivo fue realizar un proyecto de aplicación de consola en C++ que simule el juego de las torres de hanoi. Debía tener las siguientes opciones para el usuario en un menú:

◉ Jugar:  Se le pregunta al usuario el número de discos y procede a iniciar el juego con la cantidad de discos seleccionada.

◉ Instrucciones:  Se presenta el objetivo del juego, las restricciones de movimiento que hay y los controles necesarios para jugar.

◉ Créditos: Nombre de los creadores, materia para la cual fue elaborado el proyecto y nombre de la institución.

◉ Salir: Se cierra el programa.
